# 📚 Library Management System

A simple Library Management System built using **Python (OOP)** and **Tkinter GUI**.

This project demonstrates object-oriented programming concepts such as:

- ✅ Encapsulation
- ✅ Inheritance
- ✅ Polymorphism
- ✅ File Handling
- ✅ GUI Development with Tkinter

---

## 🚀 Features

### 👤 User Roles
- Student
- Admin

### 📖 Book Management
- View all books
- Borrow books
- Return books
- Add new books (Admin only)

### 💾 Data Persistence
- Books are saved to a file
- Data is automatically loaded when the program starts

---

## 🛠 Technologies Used

- Python 3
- Tkinter (GUI)
- Object-Oriented Programming
- File Handling
